# sgit-privacy-live

**PROJEKT EINGESTELLT - 13.01.2026**

---

## Was war sgit-privacy-live?

Eine privacy-fokussierte Linux-Distribution basierend auf Debian 13 (Trixie), entwickelt von [sgit.space](https://sgit.space).

## Warum eingestellt?

Die Build-Infrastruktur hatte nicht genug Ressourcen fuer den ressourcenintensiven ISO-Build-Prozess (squashfs-Komprimierung). Die CPU-Last war zu hoch fuer den parallelen Betrieb anderer Projekte auf dem Proxmox-Host.

## Was wurde erreicht?

- Vollstaendige technische Spezifikation
- Build-VM eingerichtet (Debian 13, live-build)
- Erstes bootbares ISO (v0.2.1, 1.8 GB)
- Privacy-Stack Konzept (Kill Switch, DNS-Verschluesselung, MAC-Randomisierung)
- Developer Experience Konzept (Claude Code, Starship, Zsh, Cockpit)
